Chapter 1

Why Static Spatial Weights Fail

The core assumption we need to challenge
Current model — frozen for the entire study period
$$\mathbf{F}^*_{t-1} = \mathbf{W}_{\text{static}} \cdot \mathbf{Y}^*_{t-1}$$
$\mathbf{W}_{\text{static}}$ computed once from inverse Haversine distances — same weights in 2005 and 2024
We need →
$$\mathbf{W}_t = \alpha_t \cdot \mathbf{A}_{\text{geo}} + (1-\alpha_t)\cdot\mathbf{A}^t_{\text{learned}}$$
CASA adds 265 params (0.67% of total) — updates $\mathbf{W}_t$ every time step
ENSO
El Niño weakens Walker Circulation → spatial co-variation between provinces changes sign
IOD
Positive IOD reduces moisture near Sumatra; Papua unaffected → regions decouple
Monsoon
West vs. east monsoon reverses direction of rainfall influence across islands

Stage 1

Climate Context Vector $\mathbf{c}_t$

Compressing the current climate state into a 16-dimensional summary  ($\mathbf{W}_c\in\mathbb{R}^{16\times 4}$, 80 params)
$$\mathbf{c}_t = \tanh\!\left(\mathbf{W}_c \begin{bmatrix}\bar{Y}_{t-1}\\\sigma_{Y_{t-1}}\\\text{Niño3.4}_t\\\text{DMI}_t\end{bmatrix} + \mathbf{b}_c\right), \quad \mathbf{W}_c\in\mathbb{R}^{16\times 4},\;\mathbf{b}_c\in\mathbb{R}^{16}$$
$\bar{Y}_{t-1}$
Mean rainfall — is Indonesia nationally wet or dry?
$\sigma_{Y_{t-1}}$
Spatial std dev — uniform drought vs. concentrated anomaly?
Niño 3.4$_t$
Pacific SST anomaly — Walker Circulation strength
DMI$_t$
Indian Ocean Dipole — moisture gradient toward western Indonesia
Why $\bar{Y}$ and $\sigma_Y$, not raw $Y_i$?  Including all 38 raw values would make $\mathbf{W}_c$ a $(16\times42)$ matrix — ~1,500 extra params, overwhelming CASA's 265-param budget. Mean captures national wetness; $\sigma$ captures spatial heterogeneity. Per-province detail is handled by the per-pair attention scores $e^t_{ij}$ in Stage 2a.
Stage 2a

Dynamic Attention Scores $e^t_{ij}$

Relevance of province $j$ for province $i$ at time $t$ — recomputed every month for all $N^2=1{,}444$ pairs
$$e^t_{ij} = \mathbf{v}_a^\top \cdot \text{LeakyReLU}\!\left(\mathbf{W}_a \begin{bmatrix}Y_{t-1,i}\\Y_{t-1,j}\\|Y_{t-1,i}-Y_{t-1,j}|\\\mathbf{c}_t\end{bmatrix} + \mathbf{b}_a\right)$$
Parameter definitions (Stage 2a: 168 params total)
$\mathbf{W}_a \in \mathbb{R}^{8\times 19}$ $d_a\times(3+d_c)$ Weight matrix of attention MLP — 152 params
$\mathbf{b}_a \in \mathbb{R}^8$ $d_a$ Bias of attention MLP — 8 params
$\mathbf{v}_a \in \mathbb{R}^8$ $d_a$ Projection vector — learnable parameter that collapses the 8-dim hidden representation into one scalar score $e^t_{ij}$. Acts as a learned "relevance question" asked of each province pair — 8 params
$\gamma = 0.2$ scalar LeakyReLU negative slope — prevents dead neurons; follows GAT convention (Veličković et al. 2018)
Why LeakyReLU?
Plain ReLU: gradient=0 for $x\leq0$ → neurons die. LeakyReLU: gradient=$\gamma=0.2$ → always nonzero → no dead neurons.
Softmax → $\mathbf{A}^t_{\text{learned}}$
$$\tilde{\alpha}^t_{ij} = \frac{\exp(e^t_{ij})}{\sum_{k=1}^N \exp(e^t_{ik})}$$
Full $N\times N$ scope — row-stochastic by construction.
Stages 2b & 3

Blending Gate $\alpha_t$ & Final $\mathbf{W}_t$

$\alpha_t$ answers: how much should we trust geography right now?  ($\mathbf{w}_\alpha\in\mathbb{R}^{16}$, $b_\alpha\in\mathbb{R}$ — 17 params)
$$\alpha_t = \sigma\!\left(\mathbf{w}_\alpha^\top \mathbf{c}_t + b_\alpha\right) \in (0,1)$$
Normal / Neutral
$\alpha_t \approx 0.7$–$0.9$
Geography dominates — spatial patterns as expected
Moderate El Niño
(Niño 3.4 ≈ 1.2)
$\alpha_t \approx 0.3$–$0.5$
Learned patterns become significant; teleconnections emerge
Strong El Niño
(Niño 3.4 ≥ 1.8)
$\alpha_t \approx 0.1$–$0.3$
Teleconnection patterns dominate over geographic proximity
$$\mathbf{W}_t = \alpha_t\cdot\mathbf{A}_{\text{geo}} + (1-\alpha_t)\cdot\mathbf{A}^t_{\text{learned}} \qquad \mathbf{F}^*_{t-1} = \mathbf{W}_t\cdot\mathbf{Y}^*_{t-1}$$
Chapter 2

CASA Forward Pass — Algorithm

Complete computation at each time step $t$ — replace $\mathbf{F}^* = \mathbf{W}_{\text{static}}\cdot\mathbf{Y}^*$ with this
-- Stage 1: Climate context (16-dim summary) Ȳ ← mean(Yt−1)   σ_Y ← std(Yt−1) c_t ← tanh(W_c · [Ȳ, σ_Y, Niño3.4t, DMIt]ᵀ + b_c) -- shape: (16,) -- Stage 2a: Attention for N² = 1,444 province pairs FOR each pair (i,j) in {1..N}²:   input_ij ← [Y_i, Y_j, |Y_i − Y_j|, c_t]ᵀ -- shape: (19,)   e_ij ← v_aᵀ · LeakyReLU(W_a · input_ij + b_a, γ=0.2) -- scalar score ã_ij ← softmax_row(e_ij) → A_learned_t : (N×N) row-stochastic -- Stage 2b: Blending gate α_t ← sigmoid(w_αᵀ · c_t + b_α) -- scalar ∈ (0,1) -- Stage 3: Combine W_t ← α_t · A_geo + (1 − α_t) · A_learned_t F* ← W_t · Y*t−1 -- adaptive spatial lag -- LSTM (unchanged) — 3 inputs concatenated x_t ← [Y*t−1, F*, X*t] → ŷ_t ← LSTM(x_t, ht−1, Ct−1)
Total CASA: 265 params = 80 (Context MLP) + 168 ($\mathbf{W}_a, \mathbf{b}_a, \mathbf{v}_a$) + 17 (Gate)  ·  265 / 39,343 ≈ 0.67% of full model
Chapter 3 · Proof A

Why GAT Produces Static Attention

Adapted from Brody, Alon & Yahav (GATv2, ICLR 2022) — motivates CASA's GATv2 ordering
Claim: In GAT, the ranking of any two key nodes $j_1$ vs $j_2$ never changes regardless of which query node $i$ asks — attention is monotone (static).
(1)GAT formula: $e_{ij}^{\text{GAT}} = \mathbf{a}^\top\,\text{LReLU}(\mathbf{W}\mathbf{h}_i \| \mathbf{W}\mathbf{h}_j)$
(2)$\mathbf{W}$ applied separately to $\mathbf{h}_i$ and $\mathbf{h}_j$ before concat → additive decomposition:
$$e_{ij}^{\text{GAT}} = \underbrace{\mathbf{a}_L^\top\,\text{LReLU}(\mathbf{W}\mathbf{h}_i)}_{g(i)} + \underbrace{\mathbf{a}_R^\top\,\text{LReLU}(\mathbf{W}\mathbf{h}_j)}_{h(j)}$$
(3) Compare $j_1$ vs $j_2$ for any query $i$: $e_{ij_1}>e_{ij_2} \iff g(i)+h(j_1)>g(i)+h(j_2) \iff h(j_1)>h(j_2)$ — no $i$ involved ■
GATv2 fix (used in CASA):  $e_{ij}^{\text{GATv2}} = \mathbf{v}^\top\,\text{LReLU}(\mathbf{W}[\mathbf{h}_i \| \mathbf{h}_j])$ — $\mathbf{W}$ sees both jointly before nonlinearity → cross-terms → truly dynamic ✓
Chapter 3 · Proofs B–D

Mathematical Guarantees of CASA

Three original propositions — consequences of the architecture design
B — $\mathbf{W}_t$ is Row-Stochastic  (Proposition 1, original)
$$\sum_j W_t(i,j) = \alpha_t\underbrace{\sum_j A_{\text{geo}}(i,j)}_{=\,1} + (1-\alpha_t)\underbrace{\sum_j \tilde{\alpha}^t_{ij}}_{=\,1} = 1 \quad \checkmark$$
$\mathbf{F}^* = \mathbf{W}_t\cdot\mathbf{Y}^*$ is a valid weighted average of rainfall — physically meaningful.
C — CASA Subsumes the Static Baseline  (Proposition 2, original)
$$b_\alpha \to +\infty \;\Rightarrow\; \alpha_t \to 1 \;\Rightarrow\; \mathbf{W}_t = \mathbf{A}_{\text{geo}} \quad\blacksquare$$
If no adaptive pattern found → optimizer drives $b_\alpha$ large → CASA reduces to static LSTM-GSTARX. ∴ train error(CASA) ≤ train error(static).
D — Bounded Deviation from Geographic Prior  (Proposition 3, original)
$$\|\mathbf{W}_t - \mathbf{A}_{\text{geo}}\|_F = (1-\alpha_t)\,\|\mathbf{A}^t_{\text{learned}} - \mathbf{A}_{\text{geo}}\|_F$$
$\alpha_t$ acts as implicit regularizer — higher $\alpha_t$ tightens geographic constraint. No explicit L2 needed.
